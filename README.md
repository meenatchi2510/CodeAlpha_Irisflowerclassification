# CodeAlpha_Irisflowerclassification
Iris classification is a supervised learning task using the famous Iris dataset. It predicts the species of iris flowers—Setosa, Versicolor, or Virginica—based on four features: sepal length, sepal width, petal length, and petal width. It’s a beginner-friendly example for testing classification algorithms like SVM, k-NN, and decision trees.
